# Yarl

Platform agnostic 2D rasterization library.

## Implemented Primitives

- Rectangle
- Triangle
- Arc
- Circle
- Line

## Primitives - TODO

- PPM
- Raylib

## Build Instructions

```console
$ git clone --depth 1 https://github.com/lukasx999/yarl.git
$ cd yarl/
$ make <flags>
```

### Available build flags:
- `static` (*default*)
- `shared`
- `test`
- `web`
